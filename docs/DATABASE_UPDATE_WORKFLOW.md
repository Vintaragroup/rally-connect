## Error Prevention Matrix

| Situation | Action | Result |
|-----------|--------|--------|
| Schema changed, DB reachable | `prisma migrate dev --create-only` | Migration file generated, no data loss |
| Schema changed, DB unreachable | Create migration file manually | Safe to commit, deploy later |
| Try to `migrate dev` without `--create-only` | Migration asks for reset | âŒ Data loss risk |
| Run `migrate deploy` with reviewed file | Applies safely | âœ“ No reset prompt, no data loss |
| Validate schema after migration | Confirms sync | âœ“ Schema = Database |
